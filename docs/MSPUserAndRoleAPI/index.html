    MSPUserAndRoleAPI

    All URIs are relative to https://use1-omada-northbound.tplinkcloud.com

    Method HTTP request Description
    CreateCustomerRole Post /openapi/v1/msp/{mspId}/account/customer-roles Create new customer role
    CreateNewMspRole Post /openapi/v1/msp/{mspId}/roles Create new msp role
    CreateNewMspUser Post /openapi/v1/msp/{mspId}/users Create new msp user
    DeleteCustomerRole Delete /openapi/v1/msp/{mspId}/account/customer-roles/{roleId} Delete an existing customer role
    DeleteMspRole Delete /openapi/v1/msp/{mspId}/roles/{roleId} Delete an existing msp role
    DeleteMspUser Delete /openapi/v1/msp/{mspId}/users/{userID} Delete an existing msp user
    GetAllCustomerRoles Get /openapi/v1/msp/{mspId}/customer-roles Get customer role list in msp
    GetAllMspCloudUsersExcludeRoot Get /openapi/v1/msp/{mspId}/users/cloud Get all MSP cloud users exclude owner
    GetAllMspLocalUsersExcludeRoot Get /openapi/v1/msp/{mspId}/users/local Get all MSP local users exclude owner
    GetAllMspRoles Get /openapi/v1/msp/{mspId}/roles Get msp role list
    GetAvailableCustomerRoles Get /openapi/v1/msp/{mspId}/roles/customer/roles/available Get available customer roles for creating new user
    GetAvailableMspRoles Get /openapi/v1/msp/{mspId}/roles/available Get available MSP roles for creating new user
    GetGridUsers1 Get /openapi/v1/msp/{mspId}/users Get msp user list
    GetMFAStatus Get /openapi/v1/msp/{mspId}/mfa/status Get the status of the MSP system's Two-Factor Authentication.
    GetMspAppGridUsers Get /openapi/v1/msp/{mspId}/all-users Get msp user list for app
    GetMspRole Get /openapi/v1/msp/{mspId}/roles/{roleId} Get msp role info
    GetUser1 Get /openapi/v1/msp/{mspId}/users/{userID} Get msp user info
    ModifyCustomerRole Put /openapi/v1/msp/{mspId}/account/customer-roles/{roleId} Modify an existing customer role
    ModifyMspMFAStatus Post /openapi/v1/msp/{mspId}/mfa/status Modify the status of the MSP system's Two-Factor Authentication.
    ModifyMspRole Put /openapi/v1/msp/{mspId}/roles/{roleId} Modify an existing msp role
    ModifyMspUser Put /openapi/v1/msp/{mspId}/users/{userID} Modify an existing msp user
    ModifyMspUserBatch Patch /openapi/v1/msp/{mspId}/users/batch-temporary-users Batch modify MSP users.
